# Bookstre

**Bookstre** is a modern web application built with **React** that allows users to browse, manage, and track books. It uses **Firebase** for authentication and real-time database management, ensuring a smooth, secure, and scalable experience.

---

## Features

- **Firebase Authentication** — Secure login and signup using Firebase Auth.
- **Book Management** — Add, edit, and delete books from your personal library.
- **Cloud Database** — All user data and book entries are stored and synced via Firebase Firestore.
- **Real-Time Updates** — Instantly reflects changes made to your book collection.
- **Modern UI** — Built with React and styled for a clean, responsive experience.
- **Search & Filter** — Easily find books in your collection.

---

## Tech Stack

- **Frontend:** React
- **Backend / Database:** Firebase Firestore
- **Authentication:** Firebase Auth
- **State Management:** React Hooks / Context API
- **Styling:** Tailwind CSS
